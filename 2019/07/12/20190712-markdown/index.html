Markdown入门 (一)

发表于 2019-07-12 阅读次数：

¶| 前言

想写博客好久了，但是因为自己懒 😂，一直拖。这两天终于下定决心要开通博客，记录自己的学习心路。今天简单介绍一下Markdown语言，也就是这篇博客的使用语言。

¶| Markdown 前世今生

Markdown是一种轻量级标记语言，创始人为约翰·格鲁伯（英语：John Gruber）。它允许人们“使用易读易写的纯文本格式编写文档，然后转换成有效的XHTML（或者HTML）文档”。

Markdown对于图片，图表、数学式也有很好的支持，因此许多网站都广泛使用 Markdown 来撰写帮助文档或是用于论坛上发表消息。最著名的莫过于广大程序猿熟悉的 GitHub，Stack Overflow 等等。

¶| 写在最后

写博客的原因是想以此来督促自己，也是对自己学习的一个总结。只有把别人教会了，你才算是真正学会了。第一篇博客算是开张了，希望接下来再接再厉，通过点滴的积累让自己慢慢强大起来。最后以自己的座右铭作为结束语：
